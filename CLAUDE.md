This repository contains RFCs (Requests for Comment) for standardizing AI agent tooling
conventions across engineering teams, with a focus on Claude Code and compatible IDE tools.

- Propose and refine standards for AI coding assistant configuration

- Provide test harnesses for empirical validation of tool support
